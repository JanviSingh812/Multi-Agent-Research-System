# Multi-Agent Research System

A powerful, asynchronous multi-agent research pipeline built using **LangChain** and **Streamlit**. 

## 🚀 Features
- **Search Agent:** Uses Tavily API to gather up-to-date web information entirely for free.
- **Reader Agent:** Extracts and scrapes detailed content from chosen source URLs.
- **Writer & Critic Chains:** Automatically drafts and refines a comprehensive final research report.
- **LLM Backbone:** Powered entirely by open-source infrastructure running **Llama 3.3** via **Groq** for blazing-fast, cost-free inference.
